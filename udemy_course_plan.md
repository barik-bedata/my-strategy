## Content Strategy — Same vs Different

### ✅ Same (UI/Template — Udemy does NOT count this)
- Slide layout & design template
- Font family & color theme
- Intro / Outro template
- Thumbnail design style
- Logo / branding

### 🔄 Different (Actual Content — must vary per language)
- **Code implementation** → C++ / Java / Python syntax
- **Code explanation** → Language-specific walkthrough
- **Dry run** → Different examples & data structures
- **Intuition framing** → Reworded per language mindset
- **Slide text** → Rewritten, not copy-pasted
- **Language-specific tips:**
  - C++ → STL, vectors, pointers, competitive programming angle
  - Java → Collections framework, OOP thinking, FAANG tips
  - Python → Pythonic approach, built-ins, LeetCode tricks
