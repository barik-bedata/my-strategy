# 🎬 DSA Animation & Video Production Master Blueprint

এই ডকুমেন্টটি ডার্ক মোড (`#121212`) ব্যাকগ্রাউন্ডের ওপর ভিত্তি করে তৈরি একটি প্রিমিয়াম, হাই-রিটেনশন এবং ইন্টারন্যাশনাল স্ট্যান্ডার্ডের DSA (Data Structures & Algorithms) কোর্স প্রোডাকশন গাইডলাইন। লার্নিং সাইকোলজি (Learning Psychology), কগনিティブ সায়েন্সের নিয়মনীতি এবং মিনিমালিস্ট মোশন ডিজাইন ব্যবহার করে স্টুডেন্টদের কগনিティブ লোড কমানো এবং এনগেজমেন্ট বাড়ানোই এই গাইডের মূল উদ্দেশ্য।

## 🎨 1. Global Color Psychology & Mental Model
পুরো কোর্স জুড়ে প্রতিটি ডেটা স্ট্রাকচারের প্রতিটি নোড, এজ, পয়েন্টার এবং টেক্সটের স্টেট (State) এই গ্লোবাল কালার কোড মেনে চলবে। মস্তিষ্ক এই প্যাটার্নটি ট্রাফিক লাইটের মতো অটোমেটিক্যালি চেনে, তাই এটি পুরো কোর্সে ১০০% কনসিস্টেন্ট রাখতে হবে।

| কালার/রঙ | হেক্স কোড (`Hex`) | সাইকোলজিক্যাল রোল (Role in DSA) | লার্নারের সাবকনশাস রেসপন্স |
| :--- | :--- | :--- | :--- |
| **Background** | `#121212` | Matte Black / আই-কমফোর্ট ডার্ক মোড | ক্লান্তিহীন দীর্ঘক্ষণ দেখার প্রফেশনাল বেস। |
| 🔵 **BLUE** | `#1A73E8` | Neutral / Scanning / Processing | "এখন দেখছি/প্রসেস করছি, লজিক এখনো মেলেনি।" |
| 🟢 **GREEN** | `#198754` | Success / Correct / Target Found | **"Bingo! এটাই উত্তর / কোড Accepted।"** |
| 🔴 **RED** | `#DC3545` | Error / Not Found / Wrong / Null | "না, এখানে নেই / নাল পয়েন্টার / কন্ডিশন ব্রেক।" |
| 🟡 **YELLOW** | `#EAB308` | Active Focus / Current Step / Pointer | "এখন ঠিক এই লাইনে বা এই নোডে কাজ হচ্ছে।" |
| 🟠 **ORANGE** | `#F97316` | Warning / Already Visited / Backtrack | "এই নোড/রাস্তা আগে দেখা হয়েছে, এটা সাকসেস না।" |
| ⚪ **WHITE** | `#E0E0E0` | Static / Label / Neutral Text | "এটা শুধু ইনফরমেশন বা আনভিজিটেড নোড।" |
| 🩶 **GRAY** | `#424242` | Inactive / Background Element | "এটা এখন relevant না, ব্রেইন স্কিপ করতে পারে।" |

### ⚠️ গুরুত্বপূর্ণ কারেকশন নোট (Strict Rule):
> **Visited State-এ কখনো GREEN ব্যবহার করা যাবে না।** গ্রিন দেখলে ব্রেইন অবচেতনভাবেই ধরে নেয় টার্গেট মিলে গেছে। তাই Tree, Graph বা Array—সব জায়গায় Visited নোড বা ব্যাকট্র্যাকিং পাথের রঙ হবে **ORANGE**। **GREEN** শুধুমাত্র ফাইনাল আনসার বা সাকসেসফুল সাবমিশনের জন্য সংরক্ষিত।

## 📐 2. Typography, Layout & Text Color Hierarchy
* **Font Type:** ক্লিন, মডার্ন সান্স-সেরিফ ফন্ট (যেমন: Inter, Poppins, বা কোডের জন্য Fira Code)।
* **Stroke Thickness (Edges/Borders):** ডার্ক মোড স্ক্রিনে ভিডিও রেজোলিউশন কম থাকলেও (যেমন স্টুডেন্ট যদি 480p-তে দেখে) নোড ও এজ যেন ক্রিস্প থাকে, তাই বর্ডার থিকনেস সবসময় `2px` থেকে `3px` রাখবেন।
* **Text Color Selection Rule over Colored Nodes:**
  * **Dark/Blue/Red/Green Fill:** টেক্সট কালার হবে পিওর **WHITE** (`#FFFFFF`)।
  * **Yellow/Bright Fill:** টেক্সট কালার হবে পিওর **BLACK** (`#000000`) কারণ হলুদের ওপর সাদা টেক্সট রিড করা যায় না (Contrast Ratio ব্রেক করে)।
