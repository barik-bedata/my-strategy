## Pattern Name Changes (Copyright-Safe Renames)

| Original Name | Renamed To | Reason |
|---|---|---|
| Slow & Fast Pointers | Floyd's Cycle Detection (Runner Technique) | "Slow & Fast Pointers" is directly associated with Grokking the Coding Interview branding |
| In-Place Reversal of Linked List | Linked List In-Place Manipulation | Exact phrase trademarked by Educative/Grokking |
| Island / Matrix Traversal | Grid & Matrix DFS/BFS | "Island Pattern" is a Grokking-coined term |
| Two Heaps | Dual Heap (Min-Max Balancing) | "Two Heaps" pattern name is Grokking-associated |
| K-Way Merge | Multi-List Heap Merge | "K-Way Merge" is a Grokking pattern category name |
| 0/1 Knapsack (Dynamic Programming) | Include/Exclude DP (0-1 Knapsack) | Reframed with descriptive prefix; "0/1 Knapsack" alone is a standard CS term so kept in brackets |
